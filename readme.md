# Cricket Game
# Getting Started



clone the project in your local computer
```
$ git clone https://github.com/GouravSardana/cricket-game
```




## Installing and Running the project

```
$ cd conf
```
```
$ pip install -r requirement.txt
```

```
$ cd ../src/
```

```
$ python3 main.py
```
